# Design Brief — שני תקצירי HTML לאזה"ת חולון

> **Brief עצמאי להעלאה לכלי עיצוב.** מכיל את כל ההנחיות, התוכן ושפת העיצוב הדרושים. אין צורך בהקשר נוסף.

---

## 1. המטרה

לייצר **שני קבצי HTML נפרדים** המסכמים את דוח איכות מי-התהום באזה"ת חולון (מאי 2026). שניהם נשענים על **אותו גוף ממצאים**, אך נבדלים בקהל, בעומק הטכני, באופן ייחוס מקורות הזיהום ובטון.

- **לא** דף הבהרה/פלייר; **לא** גרסת עיתונות דרמטית.
- **לא** ברמת "האם המים מהברז שלי בסדר" — הקהל הציבורי הוא **קהל מתעניין בעל רקע** (אקדמיה, ארגוני סביבה, אנשי מקצוע, תושב מעורה).

**תוצר**: HTML בלבד · RTL עברית · ללא JS חיצוני (HTML/CSS/SVG טהור) · ידידותי להדפסה (A4).

## 2. שני התוצרים

| | **תקציר מנהלים — פנים-ארגוני** | **גרסה לציבור המתעניין** |
|---|---|---|
| **קובץ** | `HOLON_EXECUTIVE_SUMMARY_INTERNAL.html` | `HOLON_EXECUTIVE_SUMMARY_PUBLIC.html` |
| **קהל** | הנהלת רשות המים, מנהלי סביבה, מקבלי החלטות | בעלי רקע — אקדמיה, ארגוני סביבה, אנשי מקצוע, תושב מעורה |
| **עומק טכני** | מלא (µg/L, % תקן, מגמות, ודאות) | מושגי יסוד; "פי כך מהתקן" במקום z-score |
| **ייחוס מקורות** | שמות מתקנים אמיתיים + ודאות (HIGH/MEDIUM/LOW) | **גנרי בלבד** ("מתקן ביטחוני", "תחנת דלק") |
| **מטריקס החלטות** | **כן** (פנימי בלבד) | לא |
| **פערי ניטור** | אחריות ארגונית ("קידוחים שקטים 40-150 חודשים") | שקיפות ("אזורים שבהם הניטור צריך להתחזק") |
| **טון** | ישיר, פעולתי, "מה לעשות + מי" | מסביר, שקוף, לא מתנשא |

**עיקרון מנחה**: אותם נתונים, מסגור שונה. שלושה דברים זזים: (1) ייחוס מקורות, (2) עומק טכני, (3) שכבת "מה עושים" (מטריקס — פנימי בלבד).

## 3. שפת עיצוב (Design Tokens)

סגנון **טיפוגרפי-אקדמי מונוכרום** — עקבי עם דוחות הפרויקט הקיימים.

### צבעים
```css
:root {
  --paper:      #fdfcf9;   /* רקע גוף — לבן-שמנת חמים */
  --ink:        #1a1a1a;   /* טקסט ראשי, גבולות, צירים */
  --ink-2:      #3a3a3a;   /* טקסט גוף משני */
  --soft:       #6b6b6b;   /* תוויות, כיתובים, מטא */
  --rule-light: #cfcfcf;   /* קווי הפרדה */
  --rule-faint: #e6e3dd;   /* גבולות שורות טבלה */
  --red:        #a82a1c;   /* התרעה — חריגות, מגמות עולות, דחיפות */
  --red-soft:   #d99a93;   /* אדום מעומעם — פערי ניטור */
  --grey-1:     #f0ede6;   /* כותרות טבלה, רקע KPI */
  --grey-2:     #d8d3c8;
  --grey-3:     #a8a29a;
  --grey-4:     #5a564f;   /* רקע חומרה גבוהה */
}
```
**כלל סמנטי לאדום** (`#a82a1c`): שמור לחריגות תקן, פערי ניטור, ומגמות עולות בלבד. לא לקישוט.

### פונטים (Google Fonts)
- **Frank Ruhl Libre** (serif עברי, RTL) — גוף וכותרות. 400/500/700/900.
- **Source Sans 3** (sans) — תוויות UI, כותרות טבלה, מטא. 400/600.
- **IBM Plex Mono** (monospace) — מספרים, קודים, ערכי נתונים. 400/500.

### RTL
- `<html lang="he" dir="rtl">`.
- כל אסימון LTR בתוך טקסט עברי (מספרים+יחידות, ראשי-תיבות כימיים כמו TCE/PCE/MTBE, שנים, סמלי מתכות) — לעטוף ב-`<bdi>`.
- `p, li, td, th, h1–h4, figcaption { unicode-bidi: isolate }`.

### פריסה
- עמודה מרכזית יחידה, `max-width: ~920px`, רקע `--paper`. ללא sidebar.
- רכיבים מומלצים: מאסטהד עם meta-chips · כותרת סעיף (מספר monospace + H2 serif עם קו תחתון) · **Severity Ledger** (grid 3-עמודות: משפחה / נרטיב / מספר אדום גדול ×N) · **מטריצת חומרה** (קידוחים × משפחות, תאי bucket s0–s8) · **כרטיסי KPI** (כרטיס דחוף ברקע `#fff5f3`/אדום) · **מפת ITM** (scatter לפי קואורדינטות, צבע לפי חומרה) · timeline פערי-ניטור · טבלת המלצות.

### סולם חומרה → צבע
9 רמות (0–8) = `bucket(C_max_5y / DWS × 100)`. מיפוי תאים: s0→שקוף · s2→`#f0ede6` · s4→`#d8d3c8` · s6→`#5a564f` (טקסט לבן) · s8→`#a82a1c` (אדום, טקסט לבן).

## 4. KPIs (מאסטהד שני הקבצים)

| מדד | ערך | הערה |
|---|---|---|
| קידוחי **ליבה פעילה** | ~placeholder | קידוחים שנדגמו ב-5 השנים האחרונות + נבדקו למזהמים מעבר לדלק. המספר הסופי בהכנה — השתמש ב-placeholder בעיצוב. |

## 5. שבעת הממצאים המרכזיים

דחיפות: 🔴 קריטי · 🟠 גבוה · 🟢 חיובי. לכל ממצא ניסוח **פנימי** (טכני) ו**ציבורי** (מוסבר).

| # | ממצא | דחיפות | ניסוח פנימי | ניסוח ציבורי |
|---|---|---|---|---|
| 1 | **חריגת תקן בקידוח הפקה** — מק חולון 14, TCE 7.59 µg/L = 101% (04/2025); ירד מתחת לתקן בהמשך | 🔴 | חריגה ראשונה אחרי עשור בקידוח הפקה (מקורות). נדרשת הצהרה רגולטורית + בדיקת נתיב אספקה לרשת | חציית סף בקידוח אספקה — הניטור הוגבר; המדידות הבאות חזרו מתחת לתקן |
| 2 | **DNAPL עיקש** — נד סונול המלאכה מ-1, TCE 2,722 µg/L (01/2026) | 🔴 | יציב 3 שנים למרות שיקום ISCO (2013–2020) באתר ציפוי סגור — מקור עומק שלא טופל | מוקד תעשייתי היסטורי שעדיין גבוה — חקירת מקור עומק נמשכת |
| 3 | **התפשטות 1,4-דיאוקסן** — נת חווה חקלאית א, 38.8 µg/L (1,293% מתקן EPA, 02/2026); נת חולון 11 הגיע ל-1,036 µg/L | 🔴 | פלאום מתפשט מהאשכול התעשייתי לקידוח חקלאי במרחק ק"מ. דרוש אישוש + איתור יצרני PVDC | מזהם חדש מתפשט לעבר אזורים חקלאיים — איתור המקור בעיצומו |

| 6 | **נקודה עיוורת PFAS** — ~3.6% מהקידוחים נדגמו, כולם אפס; המתקנים בסיכון גבוה לא נדגמו | 🟠 | חסר כיסוי באתרי AFFF (דלק) ובציפוי כרום (PFOS עד 2010). דרוש קמפיין 8–12 קידוחים | כיסוי הניטור ל-PFAS חלקי — דיגום ממוקד מתוכנן |

## 6. מטריקס ההחלטות (פנים-ארגוני בלבד)

טבלה דו-ממדית. **בלי ציר זמן. בלי גורמים חוץ-ארגוניים.** עמודות: קטגוריית פעילות ← פעולה ספציפית ← מה צריך (משפט) ← הערות.

| קטגוריית פעילות | פעולה ספציפית | מה צריך (משפט אחד) | הערות |
|---|---|---|---|
| **עדכון תכנית הניטור** | ניטור חודשי במק 14 | מעבר לדיגום TCE + 1,4-דיאוקסן חודשי בקידוח ההפקה שחצה תקן | חריגה 101% (ממצא 1) |
| | חידוש קידוחים שקטים | החזרת נת חולון 2 + נד המרכבה ק2 + אשכול אגד לדיגום CVOC | פער 47–74 ח' (ממצא 4) |
| | הרחבת פאנל | הוספת 1,4-דיאוקסן ותוצרי פירוק (VC, ethene) לפאנל ה-CVOC | ממצאים 3,5 |
| | ספציאציה | דיגום חוזר עם הפרדת Cr(VI)/Cr(III) בנת חולון 26 + 14 | |
| **קידוחי ניטור חדשים/חלופיים** | קמפיין PFAS | קידוח/דיגום 8–12 קידוחים באתרי סיכון (מרכבות האש, אגד, ציפוי, הפקה) | ממצא 6; תלוי תקציב |
| | קידוחים חקלאיים | דגימה חוזרת דחופה בנת חווה חקלאית א/ב | ממצא 3 |
| | ניטור עומק | קידוחי ניטור רב-מפלסיים באתרי DNAPL חשודים | ממצא 2 |
| **חקירת מקורות זיהום** | מיפוי DNAPL בעומק | חקירת מקור עומק בתדירגן/סונול, אלביט, הפלד-הסדנה | ודאות גבוהה (ממצא 2) |
| | איתור מקור 1,4-דיאוקסן | חיפוש במרשם התעשייה אחר יצרני PVDC/דיאוקסן | ראשון מסוגו (ממצא 3) |
| | בדיקת מיכלים תת-קרקעיים | בדיקת UST בכל תחנות אגד ברמת index 8 | ממצא 4 |

| | קריטריון השעיה | מדיניות השעיה זהירה לקידוחי הפקה בסיכון (מק חולון 12, 23) | |

## 7. ייחוס מקורות — פנימי מול ציבורי

פנימי: שמות אמיתיים + ודאות. ציבורי: **קטגוריה גנרית בלבד**.

| שם אמיתי (פנימי) | ודאות | מזהמים | קטגוריה גנרית (ציבורי) |
|---|---|---|---|
| תדיראן קשר / אלביט | HIGH | TCE, PCE, DCE, כרומאט, ציאנידים | מפעל אלקטרוניקה / מתקן ביטחוני (פעיל) |
| תדירגן | HIGH | TCE, PCE, כרומאט, ציאנידים | מפעל ציפוי מתכות (סגור, שוקם) |
| מחנה דלק צה"ל / סדנת נגמ"שים | MEDIUM | BTEX, MTBE, ממסים, שמנים | מתקן ביטחוני (היסטורי) |
| אגד / פז / סונול / דלק / מרכבות האש | MEDIUM | BTEX, MTBE, רכיבי דלק | תחנת דלק / חניון |
| אמקור פליז | HIGH | TCE, PCE, כספית, כרום, Cd | מפעל מתכות וציפוי (היסטורי) |
| נצח (גונן כימיקלים) | HIGH | TCE, DCE, דבקים | מפעל דבקים וכימיקלים (היסטורי) |
| לודג'יה | HIGH | כרום, צבענים | מפעל טקסטיל (היסטורי) |
| רימטל / ארן-מטל | HIGH | Ni, Al, Cu, Pb, TPH | מפעל מחזור מתכות |
| כור מתכת | HIGH | TCE, DCE, 1,4-דיאוקסן, כרום | מפעל מתכות (היסטורי) |

(במקור: 60 מתקנים — 17 HIGH, 39 MEDIUM, 4 LOW. לעיל המופיעים בנרטיב.)

## 8. מבנה כל קובץ (Layout)

### `HOLON_EXECUTIVE_SUMMARY_INTERNAL.html`
1. **מאסטהד** — "אזה"ת חולון · תקציר מנהלים · מאי 2026" · סיווג: פנימי · meta-chips.
2. **שורה תחתונה** — 3 משפטים.
3. **כרטיסי KPI** — 3–4 כרטיסים (חריגות / כיסוי PFAS / פערי ניטור).
4. **7 ממצאים** — כרטיסים צבועים לפי דחיפות; כל אחד: נתון + ודאות + ניסוח פנימי.
5. **מטריקס החלטות** — הטבלה הדו-ממדית מסעיף 6.
6. **מפת ITM** (אופציונלי) — קידוחים לפי חומרה.

### `HOLON_EXECUTIVE_SUMMARY_PUBLIC.html`
1. **מאסטהד** — "איכות מי-התהום באזה"ת חולון · 2026" · "דוח שקיפות".
2. **הקשר** — פסקה: אזור תעשייה ותיק, קידוחים בניטור, רוב המים תקינים, מוקדים היסטוריים בניטור צמוד + אזהרת המסגור.
3. **סיפור בקו-זמן** — זיהום היסטורי → שיקום שהצליח ✅ → מוקדים עיקשים → זיהום חדש מזוהה בזמן.
4. **7 ממצאים** — ניסוח ציבורי, ייחוס גנרי בלבד.
5. **מפת חום** — קידוחים לפי חומרה, **בלי שמות מתקנים**.
6. **"מה זה אומר + מה עושים"** — בלי "מי אחראי", בלי תקציב.
