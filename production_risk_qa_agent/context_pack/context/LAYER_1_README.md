## מתי להשתמש בShכבה 1?

✅ **שאלות עם צורך בהקשר**:
- "איך אנחנו יודעים שתדיראן היא מקור?" → ראה ה**קשר**
- "מה הסטטוס של תחנת אגד?" → ראה web_findings_context
- "איך השתנה הממצא בין דוחות?" → ראה reports_context
- "מה האימות ל-claim זה?" → ראה reports_context עם עדויות

## דוגמה: שאלה בShכבה 1

**שאלה**: "מי בדיוק המקור של TCE בקידוח נת חולון 11?"

**שאלה עמוקה**: "כמה בטוח שזה תדיראן, ולא תחנה אחרת?"

**כיצד למצוא בShכבה 1**:
1. `reports_context.md` — "מה הדוחות הקודמים אומרים על תדיראן?"
2. `source_candidates_context.md` (Layer 0) — "מה הראיות A/B של תדיראן?"
3. `HOLON_REPORT_V8.md` סעיף "Source Attribution" — "סיכום ממצאים מלא"

**תשובה**: "תדיראן היא HIGH confidence בגלל [A/B grades] + היסטוריה מ-[דוחות]"
