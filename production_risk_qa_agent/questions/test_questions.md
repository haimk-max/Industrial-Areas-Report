## TYPE A — ניתוח עמוק (Deep Analysis)

### A1 — ניתוח עמוק לקידוח הפקה: פ מקוה ישראל רוטשילד

**שאלה**:
```
תן ניתוח סיכון עמוק לקידוח ההפקה פ מקוה ישראל רוטשילד:
- רמת סיכון לכל משפחת מזהם בנפרד (CVOC / METALS / FUEL / PFAS)
- אילו פיצ'רים הניעו את חיזוי ה-XGBOOST (SHAP)
- רשימת קידוחי הניטור הקרובים שמהווים את "חלון" הסיכון
- ממצא פורנזי רלוונטי (decay chains / signatures בקרבת הקידוח)
```

**רובריקה (מה לבדוק)**:
- סיווג נפרד לכל משפחה: METALS = חריגת כלוריד בלבד (severity 4); CVOC/FUEL = אין חריגה כעת; PFAS = 0 בדגימה דלילה (פער).
- XGBOOST moderate (~0.55) + 3–5 פיצ'רי SHAP (כלוריד, מרחק מפלאום, היעדר CVOC כמפחית-סיכון, גורם sink).
- רשימת קידוחי-ניטור מובנית (שם + ITM + מרחק + כיוון + חריגה + מגמה + משפחה) — למשל נת חולון 14/11/2.
- פורנזיקה: הפלאום התעשייתי (TCE + decay chain) מצפון/מזרח; הכלוריד עשוי להיות חדירת-מליחות או שאיבה — להבחין.
- **דגש sink הידרולוגי**: קידוח הפקה מושך גם ממקורות שאינם במורד-זרם ישיר.

### A2 — ניתוח עמוק לקידוח הפקה לא-מנוטר: פ אזור מקור חקלאי

**שאלה**:
```
פ אזור מקור חקלאי הוא קידוח הפקה. תן ניתוח עמוק: מה הסיכון לכל משפחה,
מה אומר ה-XGBOOST, ומה הפער שצריך להדאיג אותנו?
```

**רובריקה**:
- **פער ניטור הוא הממצא המרכזי**: אין נתוני severity/trends לקידוח — לא ניתן לשלול זיהום.
- XGBOOST moderate (~0.44) עם הסבר: קרבה לחזית 1,4-dioxane + כיסוי ניטור דליל.
- חזית דיאוקסן (סמן-הובלה נייד) בקידוחי חווה חקלאית סמוכים — סימן שהפלאום מתקדם דרומה.
- אזהרה מפורשת: **אסור להסיק "נקי"** מהיעדר נתונים.
- המלצה: דגימה דחופה (CVOC + 1,4-dioxane + PFAS).

### A3 — מוקד הסיכון הגדול בתחום: נת חולון 11 (ניטור) והשלכתו על הפקה

**שאלה**:
```
נת חולון 11 מסומן very_high ב-XGBOOST. מה חומרת הממצא, ומה המשמעות
לקידוחי ההפקה במורד-זרם? פרט decay chain ופיצ'רי SHAP.
```

**רובריקה**:
- CVOC severity 8 — TCE בעשרות אלפי אחוזים מהתקן (המוקד החמור בתחום).
- decay chain TCE — 1,1-DCE — VC; 1,1-DCE גבוה = מקור ישיר (PVDC) בנוסף לפירוק ביוכימי.
- SHAP: latest_cvoc (תרומה גבוהה), severity 8, decay_chain_indicator.
- חיבור להפקה: וקטור הזרימה SW; הערכת זמן-מעבר משוער למקוה ישראל/חקלאי; הסתייגות אי-ודאות.

## TYPE B — Q&A קצר

### B1 — סיכון כללי לקידוחי הפקה *(הורץ בסבב קודם — נשמר)*

**שאלה**:
```
מהי רמת הסיכון של קידוחי ההפקה שלנו בחולון? אילו קידוחים בסכנה?
```

**רובריקה**: שני קידוחי הפקה; כרטיס לכל אחד (severity + XGBOOST + מזהמים + מקור + מגמה + פער + המלצה); דגש שהסיכון העיקרי הוא פלאום מתקרב, לא זיהום קיים בקידוח עצמו.

---

### B2 — מקורות זיהום ורמות ודאות

**שאלה**:
```
אילו מתקנים/תחנות הם המקורות הסבירים ביותר לזיהום המאיים על קידוחי ההפקה?
מה רמת הוודאות לכל שיוך?
```

**רובריקה**: רמות ודאות גבוהה/בינונית/נמוכה מפורשות; ראיות A–E; ריבוי-היפותזות (לא מקור יחיד ודאי); שמות מתקנים אמיתיים מ-source_candidates בלבד.

---

### B3 — מגמות עולות קריטיות

**שאלה**:
```
אילו קידוחים מציגים מגמה עולה שמדאיגה להפקה?
```

**רובריקה**: זיהוי קידוחים בסיווג INCREASING (למשל נת חולון 2 — PCE/כלורופורם/trans-1,2-DCE עולים); ציון ערכי Z/p מ-forensics_brief; חיבור לפער הניטור (נת חולון 2 הופסק 2022; נד המרכבה ק2 — MTBE עולה, הופסק 2021).

---

### B4 — PFAS: כיסוי ופער

**שאלה**:
```
מה מצב ה-PFAS בחולון? יש סיכון PFAS לקידוחי ההפקה?
```

**רובריקה**: כיסוי מינימלי (~4 מ-80 קידוחים); תוצאות 0.0 בנדגמים; **אסור** לומר "אין PFAS"; להציג כפער ידע; המלצת דגימה ממוקדת (דרום-מערב, כיוון הזרימה).

---

### B5 — הידרולוגיה: upgradient מול שאיבה

**שאלה**:
```
מתקן חשוד נמצא מעט במעלה-הזרם (upgradient) ביחס לקידוח הפקה.
לפי כיוון הזרימה הטבעי הוא לא אמור לזהם — האם זה נכון?
```

**רובריקה**: קידוח הפקה = sink הידרולוגי שמושך מכל הכיוונים לפי קצב שאיבה/מוליכות; הכלל "downgradient לא יכול לזהם" יתר-פשטני בהקשר שאיבה; המלצה לבדוק קצב שאיבה + מדידות בפועל.

---

### B6 — פערי ניטור קריטיים

**שאלה**:
```
באילו קידוחים חשובים הופסק הניטור, ומה ההשלכה?
```

**רובריקה**: נת חולון 2 (TCE שיא, הופסק 2022) ו-נד המרכבה ק2 (MTBE עולה, הופסק 2021); השלכה = עיוורון נתונים בשיא; המלצת חידוש דגימה דחוף.

## VALIDATION — תרחישי קצה

### V1 — קידוח הפקה **שאינו קיים** בחבילה (900 מ' דרום-מערבית לגבול)

**שאלה**:
```
שוקלים קידוח הפקה חדש מחוץ לאזור הרשום:
- מיקום: ITM 179,500 / 655,500 (כ-900 מ' דרום-מערבית לגבול אזה"ת חולון, במורד-זרם)
- אין רישום ב-zone_wells.csv ואין נתוני ניטור
- האם צפוי סיכון CVOC תעשייתי (TCE/PCE) ו-PFAS? כיצד תדווח?
```

**רובריקה**:
- סימון מפורש: **"ניבוי מבוסס מיקום/זרימה — לא מאומת בנתונים, אין מדידות בקידוח".**
- הערכה לפי מרחק + כיוון (דרום-מערב = במורד-זרם מהמוקדים — סיכון CVOC ממשי) + פורנזיקה (decay chains במעלה).
- PFAS: לא ניתן להעריך מהנתונים (פער כלל-אזורי) — לא לשלול.
- המלצה: קידוח-ניסוי + דגימה מלאה טרם הפעלה.

### V2 — ממצא שלילי: האם "נקי" באמת בטוח?

**שאלה**:
```
פ אזור מקור חקלאי — XGBOOST נמוך-בינוני וכל המדידות שיש 0.0.
אפשר להכריז שהקידוח בטוח?
```

**רובריקה**: לא להכריז "בטוח"; להבחין בין "אין חריגה שנמדדה" ל-"אין זיהום"; הקידוח downgradient + חזית דיאוקסן מתקרבת + ניטור דליל; המלצת ניטור רבעוני.

### V3 — קידוח הפקה חדש **בתוך הליבה התעשייתית**

**שאלה**:
```
שוקלים קידוח הפקה חדש בלב אזה"ת חולון, ~150 מ' מ-נת חולון 11.
מה רמת הסיכון הצפויה לפני קידוח?
```

**רובריקה**:
- סיכון צפוי **גבוה מאוד** — קרבה למוקד TCE החמור בתחום (נת חולון 11, severity 8).
- ניבוי מבוסס-קרבה (לא מאומת) + הסתמכות על מדידות שכנות כ-proxy.
- אזהרה: קידוח הפקה בלב מוקד DNAPL ימשוך זיהום אקטיבית (sink).
- המלצה: להימנע / לחלופין דגימת-עומק טרם החלטה.
