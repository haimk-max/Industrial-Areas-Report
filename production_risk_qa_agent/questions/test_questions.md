
## שאלה 5: Downgradient vs Pumping —הידרולוגיה

**שאלה**:
```
מתקן בחולון 26 שלנו קרוב ל-קידוח נת חולון 26 (8,072% Cr, מוקד קריטי).
אך המתקן הוא מעט UPGRADIENT ל-קידוח (לפי מפה). האם עדיין סיכון?
```

**צפי**:
- תשובה: "כן, סיכון משמעותי למרות downgradient. קידוח הפקה הוא sink הידרולוגי שמושך מים וזיהום מכל הכיוונים."
- הסבר: "משוך הידרולוגי תלוי בקצב משיכה וגיאומטריה אקוויפר, לא רק בכיוון הזרימה הטבעי."
- המלצה: "בדוק קצב משיכה של קידוח הפקה, ערוך דוגמה דינאמית, ובדוק אם Cr מזוהה בקידוח עצמו (כן — 4,036 µg/L). זה גם עדות שזיהום מגיע."

## שאלה 8: XGBOOST vs Deterministic — אם הם חלוקים

**שאלה** (if XGBOOST provided, not in SAMPLE):
```
XGBOOST חוזה "very high" עבור קידוח X, אך החומרה הדטרמיניסטית היא "moderate".
מי צודק? מה אני עושה?
```

**צפי**:
- "תן דעיפ (weighting) לשניהם, אבל דגל בחוסר התאמה"
- "דוגמה": "XGBOOST עלול להדגיש features שאנו לא רואים בדטרמיניסטי (e.g., קרבה למקור, קצב משיכה, trend secondary)"
- "שלב הבא": "בדוק SHAP feature importance — מדוע XGBOOST חוזה high? אם זה based על 'מרחק < 30 m', זה רלוונטי. אם זה 'seasonal recharge', בדוק במונוגרפיה הידרוגיאולוגית"
- "המלצה**: "טופל שניהם כ-signals. אם XGBOOST ≠ severity — תאריך ליבדיקה נוספת (sample + dynamic model)"

## הנחיות הערכה

**לאחר הרצת הסוכן**, בדוק:

1. ✅ **העברית**: מקצועית, בלי "bucket" / "ALERT" / "HIGH" בפרוזה → **Hebrew labels אל תכנוסו**
2. ✅ **קצרות**: תשובות ישירות 2–4 משפטים + כרטיסים (לא דוח מלא)
3. ✅ **כרטיסים**: קיימים עבור כל קידוח רלוונטי; תבנית קבועה (קידוח · סיכון · מזהמים · מקור · מגמה · פער · המלצה)
4. ✅ **רמות ודאות**: כל שיוך מקור מתודיע עם רמת ודאות: גבוהה/בינונית/נמוכה
5. ✅ **ראיות**: A–E grades מובחנים (A/B = main; C = context; D/E = background)
6. ✅ **XGBOOST integration** (אם זמין): מובילים, שמוט כשצריך ("XGBOOST ≠ severity, זה יכול לשמעות...")
7. ✅ **פערי ניטור surface**: "ניטור הופסק ב-2022", "רק 4/80 קידוחים ל-PFAS", וכו'
8. ✅ **בלא עובדות מחוץ לחבילה**: כל מונח (קידוח, parameter, facility) קיים בCSV/MD
9. ✅ **מגבלות נאמרות**: "רמת ודאות בינונית בגלל...", "פער ניטור: ניטור הופסק", וכו'
