## 🎯 איך משתמשים במפה זו

**משימה חדשה?**
1. תקרא CLAUDE.md + HANDOVER.md (דקה)
2. תקרא PROCESS.md לדרישות הפתוחות (דקות)
3. סדר קרא SSOT רלוונטי (15 דק' השר עדכן)
4. תבחן שמשהו לא היסטוריה — אם צריך עדכון מפא זו

**סשן שלך תרומה משהו?**
- עדכן PROCESS.md עם REQ חדש / סוגר
- עדכן HANDOVER.md אם נוצרה footgun חדשה
- עדכן CLAUDE.md אם שינית governance
- בסוף סשן: run `/handover` — תורם אוטומטי

**תרומה 18 אזורים?**
- בדוק כל מפרט SSOT — זה החוזה שלך
- אם צריך שינוי כללי → שנה SSOT + תעד בPROCESS
- אם צריך זח zone-ספציפי → כתוב ב-HANDOVER כfootgun
